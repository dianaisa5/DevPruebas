# Paginaweb

Descripción
- **Proyecto**: Sitio web estático simple con la página principal `pagina.html`.
- **Propósito**: Demo / pruebas locales de una página HTML.

Estructura del repositorio
- `pagina.html` — Página principal del proyecto.
- `readme.md` — Este archivo con instrucciones básicas.

Uso
- Abrir `pagina.html` directamente en el navegador (doble clic).
- O servir la carpeta localmente (opcional) con Python 3 desde PowerShell:


Luego abrir en el navegador: `http://localhost:8000/pagina.html`

Notas
- Edita `pagina.html` para cambiar el contenido de la página.
- Cambia la licencia o añade colaboradores según necesites.
